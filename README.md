# E-Commerce-Sales-Analysis-System
Designed and analyzed an E-commerce database using SQL to generate business insights.

This project focuses on designing and analyzing an E-commerce database using SQL. It simulates a real-world online store by managing customers, products, and orders, and extracts meaningful business insights through analytical queries.

**Key Features**:
1) Designed relational database with multiple tables:
    i)Customers
    ii)Products
    iii)Orders
    iv)Order Details
2)Implemented Primary Keys and Foreign Key relationships
3)Inserted structured sample data for realistic analysis
4)Performed data analysis using SQL queries

**Business Insights Generated**:
1)Total Sales Revenue
2)Top Selling Products
3)Customer-wise Spending Analysis
4)City-wise Revenue Distribution
5)Monthly Sales Trends

**Technologies Used**:
1)SQL (MySQL)
2)Relational Database Design
3)Data Analysis using Joins & Aggregations

**Key SQL Concepts Used**:
1)JOINs (INNER JOIN)
2)GROUP BY & Aggregation Functions (SUM, COUNT)
3)ORDER BY & LIMIT
4)Foreign Key Constraints
5)Data Filtering & Sorting

**Outcome**:
Built a complete end-to-end SQL-based data analysis system that helps in understanding customer behavior, sales performance, and product demand.
