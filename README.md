# E-Commerce-Sales-Analysis-System
Designed and analyzed an E-commerce database using SQL to generate business insights.

This project focuses on designing and analyzing an E-commerce database using SQL. It simulates a real-world online store by managing customers, products, and orders, and extracts meaningful business insights through analytical queries.

**Key Features**
Designed relational database with multiple tables:
    Customers
    Products
    Orders
    Order Details
Implemented Primary Keys and Foreign Key relationships
Inserted structured sample data for realistic analysis
Performed data analysis using SQL queries

**Business Insights Generated**
Total Sales Revenue
Top Selling Products
Customer-wise Spending Analysis
City-wise Revenue Distribution
Monthly Sales Trends

**Technologies Used**
SQL (MySQL)
Relational Database Design
Data Analysis using Joins & Aggregations

**Key SQL Concepts Used**
JOINs (INNER JOIN)
GROUP BY & Aggregation Functions (SUM, COUNT)
ORDER BY & LIMIT
Foreign Key Constraints
Data Filtering & Sorting

**Outcome**
Built a complete end-to-end SQL-based data analysis system that helps in understanding customer behavior, sales performance, and product demand.
